# Business Control AI

> SaaS-платформа управления бизнесом на базе ИИ-агентов

## 🎯 О проекте

Business Control AI — это операционная система бизнеса, где ИИ-агенты анализируют данные, выявляют проблемы, предлагают управленческие решения и контролируют исполнение.

**Цель**: Сокращение ФОТ и управленческой нагрузки предпринимателя за счёт автоматизации управленческих и аналитических функций.

## ✨ Возможности

### ИИ-агенты
- **Business AI** — главный агент: агрегирует данные, формирует решения
- **Finance AI** — анализ ДДС, прогноз кассовых разрывов, контроль дебиторки
- **Operations AI** — контроль загрузки техники, выявление простоев

### Модули
- 📊 **Дашборд** — единый экран с ключевыми метриками и рекомендациями ИИ
- 💰 **Финансы** — доходы, расходы, прогнозы, дебиторка/кредиторка
- ⚙️ **Операции** — проекты, техника, загрузка ресурсов
- 🧠 **Центр решений** — управленческие выводы и рекомендации ИИ
- 🏗️ **Активы** — учёт и контроль техники и оборудования

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- npm или yarn

### Установка

```bash
# Клонировать репозиторий
git clone <repository-url>
cd business-control-ai

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Docker

```bash
# Собрать и запустить
docker-compose up --build

# Или только собрать образ
docker build -t business-control-ai .
docker run -p 3000:3000 business-control-ai
```

## 🛠️ Технологии

## 📁 Структура проекта

```
src/
├── app/                    # Next.js App Router страницы
│   ├── dashboard/          # Личный кабинет
│   │   ├── finance/        # Финансовый модуль
│   │   ├── operations/     # Операционный модуль
│   │   ├── decisions/      # Центр решений ИИ
│   │   └── assets/         # Управление активами
│   └── api/                # API endpoints
├── components/             # Переиспользуемые компоненты
├── lib/                    # Утилиты и хелперы
├── services/               # Сервисы для работы с API
├── agents/                 # ИИ-агенты
├── types/                  # TypeScript типы
└── hooks/                  # React хуки
```
